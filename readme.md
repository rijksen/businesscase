# Savings Calculator

A web-based tool (HTML, CSS, JavaScript) that enables users to calculate various KPIs based on customer-provided information. The application evaluates consumption, production, and maintenance costs, and offers insights into potential savings using three adjustable parameters.

---

## Features
- **Customer data input:**
  - Annual gas consumption
  - Annual electricity consumption
  - Annual production (in tons)
  - Annual maintenance costs

- **KPI calculations:**
  - Energy consumption per ton
  - Gas consumption per ton
  - Maintenance costs per ton
  - Energy costs per ton

- **Savings simulation:**
  - Three independent savings parameters (in percentages)
  - Calculates potential savings for:
    - Gas consumption
    - Electricity consumption
    - Maintenance costs

- **PDF report generation:**
  - Generate a downloadable PDF summary containing input data, all calculated KPIs, and savings results.
---

## Technologies Used
This project is built entirely with static web technologies:
- **HTML5** – Application structure
- **CSS3** – Styling and layout
- **JavaScript** – Core logic for KPI and savings calculations

---

## Customization & Extension
Want to extend the tool?
- Add new KPIs in `app.js`
- Modify or extend PDF in `export.js`
- Adjust styling through `styles.css`
- Modify or extend the interface via `index.html`
